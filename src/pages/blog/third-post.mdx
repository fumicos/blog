About のコンテンツをたった1文加筆した。mdxの凄い所は、マークダウンの文書の中で普通に Javascript などを書いたりできることだと思う。
お作法は色々ありそうで、まだまだドキュメントは手放せない。

あと、重い腰をあげて Typescript にもちゃんと手を出した。Strict モードで書いていると、今まで Javascript の曖昧さに甘えていたことを思い知らされる。
でも、型宣言などは普通に便利だし、エラーが起こりにくくなるからいいね。
